## 🌟 About the Project

Relationships can be beautiful—but also confusing.

Vanessa was created to help users navigate love, mixed signals, communication struggles, toxic patterns, breakups, and emotional growth.

Whether you're unsure what to text next, wondering if someone truly likes you, or healing after heartbreak, this assistant offers reflective and supportive guidance.

## ✨ Core Features

✅ Analyze relationship behavior and mixed signals  
✅ Spot red flags / green flags  
✅ Suggest what to text next  
✅ Improve communication in relationships  
✅ Emotional support after breakups  
✅ Advice for building stronger affection and trust  
✅ Reflection-based insights for healthier decisions  

---

## 💬 Example Prompts

- Do they really like me?
- What should I text next?
- Is this relationship healthy?
- Help me heal after breakup.
- How can we grow together?

## 🎯 Purpose

To help users love smarter, communicate better, and build healthier emotional connections.
